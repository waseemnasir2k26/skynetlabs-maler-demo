# Amigos Maler — AI-Enhanced Homepage Concept (Demo)

Speculative design demo by [SkynetLabs](https://skynetjoe.com) for Amigos Maler GmbH, Olten (CH).

**3 design variants** (sticky switcher top-right):

- `index.html` — V2 Dark AI-Studio (lead)
- `v1.html` — V1 Swiss Editorial Precision
- `v3.html` — V3 Warm Craft Minimal

**Working AI features (no backend, no API keys):**

- **Offerte-Rechner** — 3-step wizard → instant CHF price range, Swiss aggregator rates (ofri/houzy/baunex), WhatsApp handoff with prefilled project summary
- **Vision Studio** — recolor an SVG room scene instantly, or upload a room photo and brush-paint walls in the browser (multiply blend keeps light/shadow). Photos never leave the device.

Pure HTML/CSS/JS, no build step. German (de-CH), Sie-form, CHF. `noindex` while spec demo.

MIT — fork to use. Beispielbilder via Unsplash; replaced with real project photos on production.
